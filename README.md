# 特点
 1. 集成中文
 2. 很少的自定义包
 3. 每日构建,追新
 4. 有21.02稳定版(更新少,官方只提交重要补丁)和最新开发版(官方修改更新多)两个版本
 5. 安装后,可以浏览器上直接在线更新官方源的包
